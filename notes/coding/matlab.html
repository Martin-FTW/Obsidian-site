Arrays and matrices

a = [a1, a2, a3]
A = [a11, a12; a21, a22]
a(i)
a([i1, i2, i3])
A(i,j)
A([i1, i2], [j1, j2, j3])
end the last index
: [1,...,end]

  • Assignment of multiple elements work as long as dimensions match
    • a(j)=[] removes the entry
    • A(:,j)=[] removes the -th column
  • Whenever possible, MATLAB will extend the array size in order to assign an object there
    a = [1,...,10] and a(20)=20 will give you a 1x20 array

2D Plotting

plot(x,y,linespec) % plots y against x given size(x)=size(y)
plot(x1,y1,linespec1,x2,y2,linespec2,...) % for multiple curves in same graph
plot(X,Y)==plot(X(:,1),Y(:,1),X(:,2),Y(:,2),...) % plots columns of Y against columns of X
plot(y,linespec) % plots y against 1:length(y)
plot(Y) % plots columns of Y against 1:length(y)
plot(z) % plot Im(z) against Re(z)
quiver(X,Y,V,W)  % vector field, draws arrows from (X_ij, Y_ij) to ((X+V)_ij, (Y+W)_ij)

With the same syntax as above, we have:
loglog(x,y)  % log y against log x
semilogx(x,y)  % y against log x
semilogy(x,y)  % log y against x
polarplot(theta,r)  % polar
fill(x,y,<color>) % fill polygon (convex hull)
bar(x,y) % bar graph, bar(y) gives one bar per element
barh(x,y) % horizontal bar graph
pie(x,{'label1','label2',...}) % pie chart, labels optional


Curve template:
t = linspace()
x = f(t)
y = g(x)
plot(x, y)

Vector field template: 
x = linspace()
y = linspace()
[X,Y] = meshgrid(x,y)
V = f(X,Y)
W = g(X,Y)
quiver(X,Y,V,W)

Plot options:  % Put after the plot(x,y) command
axis([a,b,c,d])  
% or xlim([a,b]);ylim([c,d])
% or axis square/equal
xlabel('x')
ylabel('y')
title('Title')
text(<pos_x>,<pos_y>,{'text', 'multiline'})
legend('y=f(x)')
grid on/off
box on/off

Line properties:  % replace plot(x,y) with this
plot(x, y, 'ro:', ...
	'LineWidth',3, ...
	'MarkerEdgeColor','k', ...
	'MarkerFaceColor',[.49 1 .63],...
	'MarkerSize',12)

Text options:  % replace any text function with this
legend('y=f(x)',...
	'FontSize', 16,...
	'FontWeight', 'bold',...
	'Color', 'r')
